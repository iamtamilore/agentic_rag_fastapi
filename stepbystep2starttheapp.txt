0. Prerequisites

* Docker installed
* Docker Compose installed
* Git installed

---

1. Clone project
   git clone [https://github.com/iamtamilore/agentic_rag_fastapi.git](https://github.com/iamtamilore/agentic_rag_fastapi.git)
   cd agentic_rag_fastapi

---

2. Fix common Docker issues (first time only)

If login expired:
docker logout
docker login

If ports are stuck:
docker compose down
docker network prune -f

---

3. (Important) Ollama port conflict fix

If port 11434 is busy, edit docker-compose.yml:

ports:

* "11435:11434"

Then Ollama will run at:
[http://localhost:11435](http://localhost:11435)

---

4. Start everything
   make up

---

5. Check running containers
   docker ps

You should see:

* db
* ollama
* web
* ingestion

---

6. Stop app
   docker compose down

---

Done.
